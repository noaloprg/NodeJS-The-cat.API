
# DLTCat
El objetivo es permitir a los usuarios autenticados en la plataforma obtener datos de una API externa para poblar una BD y luego relacionar los datos con los usuarios.

## Tecnologías
- NestJS
- Typescript
- Docker
- PostgreSQL
- Swagger UI
- AUTH y JWT

## Funciones
- Consuta a https://thecatapi.com/ para obtener gatos nuevos
 - Los que ya se encuentren registrados serán ignorados, asi como la raza
- COntenerización de la app en Docker y documentacion de la API en Swagger
